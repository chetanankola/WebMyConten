Notepad++ equivalent in Unix and Editing files on remote server using editor

There is no notepad++ for unix, so one can settle with gedit or vi .

Gedit can be used to access remote files via ssh just how we have a plugin for notepad++ to connect to remote server and edit files on the fly

Steps:

  1. Go to Places->Server on your Unix machine
  2. Select ssh and give the server credentials for eg: XXX.XXX.edu (wudnt need to specify port numbers)
  3. select bookmarks
  4. now you can browse files present on the remote server on your unix machine.. and use gedit to edit it.
