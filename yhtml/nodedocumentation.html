How to install nodejs on your machine:
download node from nodejs.org website
go to node dir and run the following
./configure
make
sudo make install

cmd to exit node console: process.exit(1);

How to install git Peruse your way to http://code.google.com/p/git-osx-installer/
(Here’s a direct link to the latest Git OS X Installer. Uploaded of September 17th, 2009. V1.6.4.4… Good lord. I’ll let you know when version 5 comes out. I kid, I kid.)
Open it.
Run the git-1.6.4.4-intel-leopard.pkg (A basic installer. Note: You’ll need administrative privileges.)
Complete the Installation
Open up a terminal
Change directories to the mounted image
bash$ cd /Volumes/Git\ 1.6.4.4\ Intel\ Leopard/
bash$ ls
README.txt setup git PATH for non-terminal programs.sh
git-1.6.4.4-intel-leopard.pkg uninstall.sh
bash$ ./setup\ git\ PATH\ for\ non-terminal\ programs.sh
Run the shell script (last line above)
